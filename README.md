## **Features**

- Simulated multi-node AI cluster using Docker containers  
- AI training using a simple Convolutional Neural Network on `FakeData.`  
- Automated deployment and execution of workloads across nodes  
- Monitoring of CPU and memory usage per node  
- Fully local, no AWS or cloud required
